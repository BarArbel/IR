Instructions:
Brackets: () OR: || AND: && NOT: !!




The query should be structured accurding to the followind pattern:

1. The query must contain atleast one word.
2. Following each word - except the last one in the expression - there must be an operand (&& or ||)
3. !! must come right before the word it's meant to apply to.
4. In order to take common stop-words into effect they must be place between ""

Example: ((word1 && word2) || "stop-wrod") && !! word3
